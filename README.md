Web开发技术-作业1
====

本地预览
----
1. 启动后端
    ```cmd
    $ cd web-development-homework1
    $ node src\assets\back-end.js
    ```

2. 启动前端
    ```cmd
    $ cd web-development-homework1
    $ npm start
    ```
3. 等待编译完成后转到 [http://localhost:4200](http://localhost:4200) 即可。
